senha

Senha is a password generation tool for making random passwords based on a input requirement.

Installation

Senha is released through RubyGems. This allows for an easier installation.

% gem update % gem install senha

Usage

Using senha is relatively simple. Some example usage would be:

% senha -l
kpsyzatdpu
% senha -n
3993355865
% senha -a --length 20
M;w%Bwh8z-1NwYiS,#8h
% senha -a --length 20 --count 10
