# passgen

Passgen is a password generation tool for making random passwords based on a input requirement.

# Usage

Using passgen is relativley simple. Some example usage would be:

	% passgen -l
	kpsyzatdpu
	% passgen -n
	3993355865
	% passgen -a --length 20
	M;w%Bwh8z-1NwYiS,#8h
	% passgen -a --length 20 --count 10
	:QGZz:FqV88B4CQoPgiG
	@m,obU-50uyI!#7-7l8A
	i8u7R4mwcYI%.QBsEVtW
	.LFchb;ni%#MC6#grF:,
	NoarJHX3$Nz!hcu3AIjQ
	cfrZGeG90ahfqSqPpq42
	5;1WU@%:C9ZJMktx$sUa
	$e5,6jdOSQ&zAV:H&6LS
	4&yqxcF2Rj,qiMQMlTt,
	#TyFKmzVTVer,;E-0^ro
